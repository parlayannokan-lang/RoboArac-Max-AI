# RoboAraç Max AI

RoboAraç Max AI, ESP32 ve Raspberry Pi destekli, 6x6 yapıda geliştirilen yapay zeka destekli robot araç projesidir.

## Proje Amacı

Bu projenin amacı; uzaktan kontrollü, sensör destekli, ileride GPS ve yapay zeka ile yarı otonom hale getirilebilecek modüler bir robot araç geliştirmektir.

## Genel Özellikler

- 6x6 motorlu kara aracı
- ESP32 ile APK üzerinden manuel kontrol
- BTS7960 motor sürücü sistemi
- Raspberry Pi 5 ile yapay zeka ve kamera altyapısı
- GPS ile gelecekte yarı otonom sürüş hazırlığı
- Robot kol entegrasyonu
- DJI Tello drone taşıma platformu hazırlığı
- 3D yazıcı ile üretilecek modüler şase
- PLA gövde, TPU tekerlek planı

## Sürüm Planı

### V1 - Temel Çalışan Araç

- 6 motorlu kara aracı
- ESP32 kontrol
- APK ile ileri / geri / sağ / sol kontrol
- BTS7960 motor sürücüler
- 12V güç sistemi
- Temel kablo kanalları
- 3D baskı şase

### V2 - Akıllı Araç

- Raspberry Pi 5
- Kamera ile görüntü işleme
- Nesne tanıma
- GPS modülü
- Mesafe sensörleri
- Pan-tilt kamera sistemi

### V3 - Gelişmiş Sistem

- DJI Tello taşıma alanı
- Yarı otonom sürüş
- Su üstü kullanım hazırlığı
- Daha gelişmiş yapay zeka karar sistemi
- Yarışma ve sunum seviyesinde proje dokümantasyonu
